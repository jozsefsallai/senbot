# senbot

## Requirements

- Node 16 or higher
- Discord bot token
- Nakiri API key (optional)

## Getting Started

1. Clone the repo

```sh
git clone git@github.com:jozsefsallai/senbot.git
cd senbot
```

2. Configure the env

```sh
cp .env.example .env
vim .env
```

3. Install dependencies

```sh
npm i
```

4. Build the bot

```sh
npm run build
```

5. Run the bot

```sh
npm run start
```
